# 七彩果坊水果定价计算器 - 微信小程序

## 项目简介

这是一个专为水果批发商和零售商设计的价格计算工具，帮助用户根据采购成本、损耗率、运输距离等因素，快速计算出合理的销售价格。

## 功能特点

- 🍎 支持23种常见水果的自动损耗率设置
- 📦 支持3种标准筐类型（4.5斤筐、5斤筐、2.5斤筐）
- 📊 智能计算毛单价、净单价、损耗后成本、运输成本、总成本
- 💰 自动生成建议开团价和线下售价
- 📈 市场竞争力分析和建议
- 📱 微信小程序界面，操作简便

## 使用方法

### 1. 开发环境准备

1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号，获取AppID

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目目录
4. 填入你的AppID（在`project.config.json`中修改`appid`字段）

### 3. 配置AppID

在`project.config.json`文件中，将`appid`字段修改为你的小程序AppID：

```json
{
  "appid": "你的小程序AppID"
}
```

### 4. 运行项目

1. 在微信开发者工具中点击"编译"
2. 在模拟器中测试功能
3. 确认无误后可以上传代码

## 功能说明

### 输入参数

- **水果种类**: 选择水果类型，系统会自动设置对应的平均损耗率
- **筐类型**: 选择包装类型（4.5斤筐、5斤筐、2.5斤筐）
- **标准重量**: 水果的标准重量（千克，默认10千克）
- **筐重**: 包装的重量（斤，根据筐类型自动设置）
- **毛单价**: 包含包装的水果单价（元/斤）
- **净单价**: 去除包装后的水果单价（系统自动计算）
- **损耗率**: 运输和储存过程中的损耗百分比
- **运输距离**: 从采购地到销售地的距离（公里，默认6公里）
- **市场参考价**: 当地市场的参考价格（可选）
- **运输成本**: 每斤每公里的运输成本（默认0.05元）
- **利润率**: 期望的利润率百分比（默认25%）

### 计算结果

- **标准重量**: 水果的标准重量（千克）
- **筐重**: 包装重量（斤，自动设置）
- **毛单价**: 包含包装的单价
- **净单价**: 去除包装后的单价
- **损耗后成本**: 考虑损耗后的实际成本
- **运输成本**: 根据距离计算的运输费用
- **总成本**: 损耗后成本 + 运输成本
- **建议开团价**: 自动调整为以.9结尾的价格
- **建议线下售价**: 开团价 + 2元
- **市场建议**: 与市场价格的竞争力分析

## 净单价计算公式

净单价 = (毛单价 × 标准重量) ÷ (标准重量 - 筐重)

其中：
- 标准重量：用户输入，默认10千克
- 筐重：根据筐类型自动设置

## 注意事项

1. 所有价格计算基于"元/斤"单位
2. 损耗率会根据选择的水果种类自动填充，可手动调整
3. 建议开团价会自动调整为以.9结尾的价格
4. 市场参考价为可选输入，用于竞争力分析
5. 运输成本默认为0.05元/斤/公里，可根据实际情况调整
6. 净单价会根据毛单价、标准重量和筐重自动计算
7. 筐重会根据选择的筐类型自动设置
8. 采购成本使用净单价进行计算
9. 标准重量默认为10千克，可根据实际情况调整
10. 运输距离默认为6公里，可根据实际情况调整
